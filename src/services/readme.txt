    Services.

A pasta "services" contem codigos necessarios para acessar serviços externos, 
vindo de outras aplicações ou servidores.


----------------------------------------////////////------------------------------------------////////////-------------------------

    Acessando Apis.

Para acessar outras aplicações ou servidores, é necessario realizar o download 
do "Axios" uma biblioteca propria para esta finalidade.

- yarn add axios.

Dentro do arquivo responsavel pelas conexões a serviços externos, basta realizar 
o importe da biblioteca "axios", assim tera acesso a todos o metodos disponiveis,
para ralizar todas as configurações necessarios para conexões a serviços externos.

Dentro do arquivo principal da aplicação, realize o import, das configurações 
do "services" assim a aplicação terá acesso a serviços de servidores externos.
