    Criando um projeto React Js.

Primeiramente realize o dowload das dependencias do react, 
para isso basta apenas abrir o cmd.

- npm install -g create-react-app

Esta biblioteca ja vem com arquivos responsaveis por fazer 
a transpilação do codigo JS, para versões mais antigas da linguagem.

Feito o dowload das dependencias, pode se usar a biblioteca recem adicionada 
para que ela crie toda a estrutra do projeto com o React, atraves do CMD.

- create-react-app "NomeDoProjeto"

----------------------------------------////////////------------------------------------------////////////-------------------------
    Pastas.

SRC.
A pasta contem todos os arquivos que contem codigos importantes 
para a aplicaçao, todos os codigos ficaram dentro desta pasta.

Public.
A pasta contem todos os arquivos que vão ser acessiveis 
pelo usuario final da aplicação.
